Introduction to Java

Java is a popular object-oriented programming language used for building a wide range of applications. It is known for its platform independence and robustness. 

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Java Variables

Variables in Java are used to store values of different types. They provide a way to reference and manipulate data within a program. 

int myNumber = 42;
String myName = "John Doe";

Java Control Flow

Java provides control flow statements such as if-else, loops, and switch-case to control the execution flow of a program based on conditions and iterations. 

if (condition) {
    // code block
} else {
    // code block
}

Java Methods

Methods in Java are reusable blocks of code that perform specific tasks. They allow code to be organized, modular, and promote code reuse. 

public int sum(int a, int b) {
    return a + b;
}

Java Classes

Classes in Java are blueprints for creating objects. They define the properties and behaviors that objects of the class can exhibit. 

public class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

Java Exceptions

Exceptions in Java are used to handle unexpected or exceptional situations that occur during the execution of a program. They help in handling errors and maintaining program flow. 

try {
    // code that may throw an exception
} catch (Exception e) {
    // code to handle the exception
}
