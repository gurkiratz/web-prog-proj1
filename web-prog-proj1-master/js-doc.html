Introduction to JavaScript

JavaScript is a high-level, interpreted programming language that allows you to add interactivity and dynamic behavior to web pages. It is one of the core technologies of the web. 

alert('Hello, World!');

JavaScript Variables

Variables in JavaScript are used to store values. They provide a way to reference and manipulate data within a program. JavaScript variables are dynamically typed, which means you can assign values of different types to the same variable. 

let myNumber = 42;
const myName = 'John Doe';

JavaScript Data Types

JavaScript has several built-in data types, including numbers, strings, booleans, objects, arrays, and more. Understanding data types is crucial for working with JavaScript variables and performing operations on them. 

let age = 25;
let name = 'John Doe';
let isStudent = true;

JavaScript Operators

JavaScript operators are used to perform operations on variables and values. They include arithmetic, assignment, comparison, logical, and more. Operators allow you to manipulate and combine data in JavaScript. 

let a = 5;
let b = 10;
let sum = a + b;

JavaScript Control Flow

JavaScript control flow statements, such as if-else, loops, and switch-case, allow you to control the execution flow of a program based on conditions and iterations. They help you make decisions and repeat actions. 

if (condition) {
    // code block
} else {
    // code block
}

JavaScript Functions

Functions in JavaScript are reusable blocks of code that perform specific tasks. They allow you to group and organize code, and they can be called and executed multiple times. Functions enable code reuse and modularity. 

function sum(a, b) {
    return a + b;
}

JavaScript Objects

Objects in JavaScript are collections of key-value pairs. They allow you to store and manipulate complex data structures. Objects can have properties and methods, which represent the characteristics and behaviors of the object. 

let person = {
    name: 'John Doe',
    age: 25,
    sayHello: function() {
        console.log('Hello, I am ' + this.name + '!');
    }
};

JavaScript Arrays

Arrays in JavaScript are used to store multiple values in a single variable. They allow you to store and manipulate collections of data. Arrays have built-in methods and properties that make working with lists of values easier. 

let numbers = [1, 2, 3, 4, 5];
let fruits = ['apple', 'banana', 'orange'];
