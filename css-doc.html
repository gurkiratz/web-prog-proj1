Introduction to CSS

CSS (Cascading Style Sheets) is a styling language used to describe the look and formatting of a document written in HTML. It enables web developers to control the visual appearance of web pages. 

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
}

CSS Selectors

CSS selectors are used to target specific HTML elements and apply styles to them. They allow you to select elements based on their tag name, class, ID, attributes, and more. 

h1 {
    color: #333;
    font-size: 24px;
}

CSS Box Model

The CSS box model describes the layout and sizing of elements on a web page. It consists of content, padding, border, and margin. Understanding the box model is crucial for creating responsive and well-structured layouts. 

.box {
    width: 200px;
    height: 150px;
    padding: 10px;
    border: 1px solid #ccc;
    margin: 10px;
}

CSS Layout

CSS layout techniques allow you to control the positioning and arrangement of elements on a web page. You can use properties like display, position, float, and more to create different layouts. 

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

CSS Flexbox

CSS Flexbox is a one-dimensional layout model that provides a flexible way to distribute space among items within a container. It simplifies the creation of responsive and flexible layouts. 

.flex-container {
    display: flex;
    flex-direction: row;
    justify-content: space-between;
}

CSS Grid

CSS Grid is a two-dimensional layout system that allows you to create complex grid-based layouts. It provides precise control over the placement and sizing of elements in both rows and columns. 

.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-gap: 10px;
}

CSS Media Queries

CSS Media Queries allow you to apply different styles based on the characteristics of the device or viewport. They are commonly used for creating responsive designs that adapt to different screen sizes. 

@media screen and (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}
